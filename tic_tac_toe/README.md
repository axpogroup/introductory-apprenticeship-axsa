# Tic Tac Toe Game
Tic Tac Toe is a simple two-player game where players take turns placing X or O on a 3×3 grid, trying to get three in a row.
It originated from the ancient Roman game “terni lapilli,” where players used stones to form lines on a grid.



## Problem Description
In Tic Tac Toe players take turns placing X or O, trying to get three in a row:
1. There is a 3x3 game board.
2. One player uses “X” and the other uses “O”.
3. Players take turns placing their symbol in an empty square.
4. The goal is to place three of their symbols in a row (horizontal, vertical, or diagonal).
5. If a player achieves this, they win the game.
6. If all squares are filled without a winner, the game ends in a draw.
